# Projeto do Site

Este repositório contém os arquivos do site exatamente como estão.

## Como executar localmente

1. Clone o repositório.
2. Entre na pasta do projeto.
3. Instale as dependências com `npm install`.
4. Inicie o ambiente de desenvolvimento com `npm run dev`.

## Observação

Nenhum conteúdo do site foi alterado neste ajuste. Apenas referências de plataforma antiga foram removidas da documentação.
